# PhishGuard — AI-Powered Phishing Email Detector

PhishGuard is a lightweight, explainable phishing email detection system 
built as a final-year Computer Science project at the University of Westminster.

## What it does
- Classifies emails as phishing or legitimate in real time
- Provides word-level explanations showing which terms influenced the decision
- Runs entirely offline with no external data transmission
